The radio is an additional control, that should be used when user
is supposed to make only one selection out of a group of options.

<DemoBox>
  <RadioDemo />
</DemoBox>

```jsx
import { Radio } from 'insites-ui'

<Radio
  id="sizeL"
  checked={isChecked}
  onChange={check}
>
  Size L
</Radio>
```
