## Component

import { Input } from 'insites-ui'
import DemoBox from '../../components/DemoBox'
import PhoneIcon from '../../assets/phone.svg'

The input is a control to get user input in a text field. 
Can be used alongside FormControl, providing an accessible label, 
validation messages, etc.

<DemoBox>
    <Input placeholder="Bryan Fury" />
</DemoBox>

```js
import { Input } from 'insites-ui'

<Input placeholder="Bryan Fury" />
```
